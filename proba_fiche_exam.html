I

Bases — Probabilité & Vocabulaire

Vocabulaire fondamental
  • Ω = ensemble de tous les résultats possibles
  • Événement A = sous-ensemble de Ω
  • Ā = événement contraire de A
  • A ∩ B = les deux se réalisent ensemble
  • A ∪ B = au moins l'un se réalise
  • A ∩ B = ∅ → A et B incompatibles
  • Ω = évt certain | ∅ = évt impossible
Axiomes de Kolmogorov
  • 0 ≤ P(A) ≤ 1 pour tout A
  • P(Ω) = 1
  • P(∅) = 0
  • P(Ā) = 1 − P(A)
  • Si A ∩ B = ∅ : P(A ∪ B) = P(A) + P(B)
  • Cas général : P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
🎯 Truc : P(Ā) = 1 − P(A) est souvent le chemin le plus rapide pour les probabilités du type "au moins un" ou "au moins une fois".
Ex : P(au moins 1 succès) = 1 − P(0 succès).
Exemple — Lancer un dé équilibré
Ω
Ω = {1, 2, 3, 4, 5, 6}
P(obtenir un pair) = P({2,4,6}) = 3/6 = 1/2
A ∪ B
A = {pair} = {2,4,6}, B = {> 4} = {5,6}
P(A ∪ B) = P(A) + P(B) − P(A ∩ B) = 3/6 + 2/6 − 1/6 = 4/6 = 2/3
A ∩ B = {6} → P(A ∩ B) = 1/6

III

Probabilité Conditionnelle & Bayes

Formule de base

P(A|B) = probabilité que A se réalise sachant que B est réalisé.

P(A | B) = P(A ∩ B) / P(B)

⚠️ Valable uniquement si P(B) > 0

Indépendance d'événements

A et B sont indépendants si la réalisation de l'un ne change pas la proba de l'autre.

A ⊥ B  ⟺  P(A ∩ B) = P(A) × P(B)

→ P(A|B) = P(A) si indépendants

🌳 Arbre pondéré + Probabilités Totales

Si {E₁, E₂, ..., En} est une partition de Ω (incompatibles, union = Ω) alors :

Formule des Probabilités Totales P(A) = P(E₁)·P(A|E₁) + P(E₂)·P(A|E₂) + ... + P(En)·P(A|En)
Méthode arbre : Multiplier les probas le long de chaque branche → additionner tous les chemins qui arrivent à A.
📝 Exemple TD — Dés truqués (100 dés, 25 truqués)
T
Branche Truqué
P(T) = 25/100 = 1/4  |  P(6|T) = 1/2  →  chemin = 1/4 × 1/2 = 1/8
T̄
Branche Normal
P(T̄) = 3/4  |  P(6|T̄) = 1/6  →  chemin = 3/4 × 1/6 = 3/24 = 1/8
Σ
Résultat
P(C) = 1/8 + 1/8 = 1/4 = 0,25 ✓
🔁 Formule de Bayes — « Remonter à la cause »
Bayes — Trouver la cause probable P(Er | A) = [P(Er) × P(A | Er)] / P(A)

Le dénominateur = P(A) calculé par les probabilités totales. On l'a déjà calculé à l'étape précédente !

📝 Exemple TD — Machines M1, M2, M3
1
Données
P(M1)=0,5 P(D|M1)=0,2 | P(M2)=0,4 P(D|M2)=0,15 | P(M3)=0,1 P(D|M3)=0
2
P(D) — Probabilités Totales
0,5×0,2 + 0,4×0,15 + 0,1×0 = 0,10 + 0,06 + 0 = 0,16
3
P(M1|D) — Bayes
P(M1)×P(D|M1) / P(D) = 0,1 / 0,16 = 5/8 = 0,625
⚠️ Piège classique : Ne pas confondre P(D|M1) = "pièce défect sachant que c'est M1" avec P(M1|D) = "c'est M1 sachant que la pièce est défect". Ce sont deux choses opposées — Bayes sert à passer de l'un à l'autre !

Variable Aléatoire Discrète (V.A.D.)

Loi de probabilité (masse)
  • f(xᵢ) = P(X = xᵢ)
  • 0 ≤ f(xᵢ) ≤ 1
  • Σ P(X = xᵢ) = 1 TOUJOURS !
  • Sert à trouver les paramètres inconnus (a, k…)
Fonction de répartition F(x)
  • F(x) = P(X ≤ x) = Σxᵢ ≤ x P(X = xᵢ)
  • F est croissante, F(−∞) = 0, F(+∞) = 1
  • P(a < X ≤ b) = F(b) − F(a)
  • P(X > a) = 1 − F(a)
📐 Espérance & Variance — Formules et propriétés
Espérance E[X] E[X] = Σ xᵢ · P(X = xᵢ)
E[X²] E[X²] = Σ xᵢ² · P(X = xᵢ)
Variance Var(X) Var(X) = E[X²] − (E[X])²
Écart-type σ σ(X) = √Var(X)

🔑 Propriétés linéaires (très souvent au DS)

E[aX + b] = a·E[X] + b

Var(aX + b) = a²·Var(X)

⚠️ La constante additive b DISPARAÎT dans Var !

Var(2X−3) = 4·Var(X)  (pas −3)

🔑 X et Z indépendantes

E[XZ] = E[X] · E[Z]

Var(X + Z) = Var(X) + Var(Z)

Var(X − Z) = Var(X) + Var(Z)

⚠️ Var(X−Z) = Var(X)+Var(Z), pas −Var(Z) !

📝 Exemple complet DS Ex.1 — Calculs E[X] et Var(X)

X prend les valeurs {0, 1} avec P(X=0)=0,3 et P(X=1)=0,7

a
E[X]
0 × 0,3 + 1 × 0,7 = 0,7
b
E[X²]
0² × 0,3 + 1² × 0,7 = 0 + 0,7 = 0,7  (attention : 0²=0, 1²=1)
c
Var(X)
E[X²] − (E[X])² = 0,7 − (0,7)² = 0,7 − 0,49 = 0,21
d
Applications linéaires
Var(2X−3) = 2²×0,21 = 0,84
E[2X+4] = 2×0,7 + 4 = 5,4
E[XZ] = E[X]·E[Z] = 0,7×0,7 = 0,49  (X,Z indép.)
Var(X+Z) = Var(X)+Var(Z) = 0,21+0,21 = 0,42
Var(X−Z+2) = Var(X)+Var(Z) = 0,42  (+2 disparaît)

Couples de Variables Aléatoires (X, Y)

📋 Tableau de contingence — lecture et calculs
X \ Yy₁y₂...Loi marginale de X
x₁p₁₁p₁₂... P(X=x₁) = somme ligne 1 →
x₂p₂₁p₂₂... P(X=x₂) = somme ligne 2 →
Loi marginale de Y P(Y=y₁) = somme col 1 ↓ P(Y=y₂) ↓ ... = 1
💡 Truc lecture tableau : Marginale de X = sommer sur les lignes (fixer x, sommer sur tous les y). Marginale de Y = sommer sur les colonnes (fixer y, sommer sur tous les x).
🔗 Tester l'indépendance de X et Y
Condition d'indépendance X ⊥ Y  ⟺  P(X=xᵢ, Y=yⱼ) = P(X=xᵢ) × P(Y=yⱼ)   pour TOUS les couples (i,j)
1
Calculer les lois marginales
Remplir la dernière ligne et la dernière colonne du tableau
2
Tester UNE case
Choisir n'importe quel (xᵢ, yⱼ) et vérifier si pij = P(X=xᵢ)×P(Y=yⱼ)
3
Conclusion
Si une seule case ne satisfait pas la condition → X et Y NE SONT PAS indépendantes
📝 Exemple DS Ex.1 — a = 0,1
Test : P(X=0, Y=1) = P(X=0) × P(Y=1) ?
✗ 0,1 ≠ 0,3 × 0,4 = 0,12
0,1 ≠ 0,12 → X et Y ne sont PAS indépendantes
Exemple où elles SERAIENT indépendantes (TD Ex.1 — X⊥Y données)
✓ P(X=1, Y=0) = 1/3 × 3/4 = 3/12 ✓
Toutes les cases satisfont la condition → X et Y indépendantes → on peut utiliser le produit pour construire la loi conjointe
📐 Covariance & Coefficient de corrélation
Covariance Cov(X,Y) = E[XY] − E[X]·E[Y]

E[XY] = Σᵢⱼ xᵢ·yⱼ·P(X=xᵢ, Y=yⱼ)
Coefficient de corrélation r r = Cov(X,Y) / √[Var(X)·Var(Y)]

−1 ≤ r ≤ +1

r = +1

Relation linéaire positive parfaite. Y augmente quand X augmente.

r = 0

Aucune corrélation linéaire (X et Y non corrélées).

r = −1

Relation linéaire négative parfaite. Y diminue quand X augmente.

Indépendant → r = 0

Si X⊥Y alors Cov=0 et r=0. Mais r=0 n'implique pas l'indépendance !

➕ Loi de Z = X + Y — Méthode
1
Lister toutes les paires (xᵢ, yⱼ)
Pour chaque case du tableau, calculer z = xᵢ + yⱼ
2
Regrouper par valeur de z
Additionner les proba de toutes les paires qui donnent le même z
3
Vérification
Σ P(Z = z) = 1 ✓

X ∈ {−2,0,1}, Y ∈ {−1,1,2}

P(Z=−1) = 0,2+0,1 = 0,3 | P(Z=0) = 0,05+0,2 = 0,25

🎯 Probabilité conditionnelle dans un tableau
P(X = x | Y = y) P(X = x | Y = y) = P(X=x, Y=y) / P(Y=y)
Exemple DS : P(X=1 | Y=1) = P(X=1, Y=1) / P(Y=1) = 0,3 / 0,4 = 3/4 = 0,75
🎯 Truc : La probabilité conditionnelle, c'est "zoomer" sur une ligne ou une colonne du tableau. On divise la case par le total de la ligne/colonne concernée.
VI

Lois de Probabilités Usuelles

LOI DE BERNOULLI  B(p)
Définition
  • Une seule épreuve
  • 2 issues : Succès (p) ou Échec (q = 1−p)
  • X ∈ {0, 1}
  • Brique de base de la binomiale
Caractéristiques
E[X] = p
Var(X) = p·q = p(1−p)
LOI BINOMIALE  B(n, p)

Nombre de succès X lors de n épreuves de Bernoulli indépendantes identiques, chacune de probabilité de succès p.

Quand l'utiliser ?
  • n épreuves répétées
  • Chaque épreuve : 2 issues uniquement
  • Épreuves indépendantes
  • Même probabilité p à chaque épreuve
  • X = nombre de succès
Formules
P(X=k) = Cnk·pk·qn−k
pour k = 0, 1, ..., n

E[X] = n·p
Var(X) = n·p·q
📝 Exemples
TD Ex.3 — Composants en panne (n=2, proba panne = p)
X ~ B(2, p)
p₂ = P(machine arrêtée) = P(X > 1) = P(X=2) = p²
Si n=4 : p₄ = P(Y > 2) = P(Y=3)+P(Y=4) = 4p³(1−p)+p⁴ = p³(4−3p)
DS Ex.2 — Chaudières sous garantie parmi 5 défectueuses
N ~ B(5, 1/41)
P(N≥1) = 1 − P(N=0) = 1 − (40/41)⁵ ≈ 1 − 0,884 = 0,116
LOI DE POISSON  Po(λ)

Modélise le nombre d'événements rares dans un intervalle de temps ou d'espace. λ = nombre moyen d'occurrences = E[X].

Quand l'utiliser ?
  • Nombre de défauts, pannes, arrivées...
  • Par unité de temps, surface, longueur...
  • λ = moyenne observée sur l'échantillon
  • X ∈ {0, 1, 2, 3, ...} → valeurs infinies
Formules
P(X=k) = (λk · e−λ) / k!

E[X] = λ
Var(X) = λ
Propriété unique : E = Var = λ
📝 Exemple TD Ex.4 — Dalles de marbre (λ = 1,2)
P(X=0) = e⁻¹·²              = 0,301
P(X=1) = 1,2 × e⁻¹·²       = 0,361
P(X=2) = (1,2)²/2! × e⁻¹·² = 0,217

P(X > 2)  = 1 − (0,301+0,361+0,217) = 0,121   ← "plus de 2 défauts"
P(X ≥ 2)  = 1 − (0,301+0,361)       = 0,338   ← "au moins 2 défauts" = qualité **
500×P(X=0) = 500 × 0,301            ≈ 150     ← dalles sans défaut
⚠️ Attention > vs ≥ : P(X > 2) exclut k=2 → on soustrait aussi P(X=2). P(X ≥ 2) inclut k=2 → on soustrait seulement P(X=0) et P(X=1).
⚡ Quand utiliser quelle loi ? — Tableau récapitulatif
LoiContexte typiqueE[X]Var(X)
Bernoulli B(p) 1 seule épreuve, 2 issues (succès/échec) ppq
Binomiale B(n,p) n épreuves indép., compter les succès, n et p connus npnpq
Poisson Po(λ) Événements rares/unité, λ = moyenne donnée λλ

⚡ Pièges & Trucs — Points de perdition

🪤 Var(X − Z) = Var(X) + Var(Z), pas −Var(Z)
✓ Toujours + pour variables indépendantes
Var(X−Z) = Var(X) + (−1)²·Var(Z) = Var(X)+Var(Z). Le signe du coefficient est mis au carré → toujours positif.
🪤 Var(aX + b) : la constante b disparaît
✓ Var(2X − 3) = 4·Var(X), pas 4·Var(X) − 3
Ajouter ou soustraire une constante ne change pas la dispersion. Seul le coefficient multiplicatif a compte (au carré).
🪤 P(X > 2) ≠ P(X ≥ 2) pour la loi de Poisson
✓ P(X > 2) = 1 − P(X≤2) | P(X ≥ 2) = 1 − P(X≤1)
P(X > k) = "strictement supérieur" = on exclut k. P(X ≥ k) = "au moins k" = on inclut k.
🪤 Confondre P(D|G) avec P(G|D)
✓ Bayes permet de passer de l'un à l'autre
P(D|G) = proba d'être défectueux SACHANT qu'on est sous garantie (donnée dans l'énoncé). P(G|D) = proba d'être sous garantie SACHANT qu'on est défectueux (question du DS → Bayes).
🪤 Oublier de vérifier que Σ probabilités = 1
✓ Toujours vérifier en fin d'exercice
Que ce soit une loi conjointe, marginale, ou la loi de Z = X+Y, la somme doit valoir exactement 1. C'est votre vérification automatique.
🪤 Indépendance : tester UNE case suffit pour conclure NON
✓ Mais vérifier plusieurs si la 1ère satisfait la condition
Si P(X=xᵢ, Y=yⱼ) ≠ P(X=xᵢ)×P(Y=yⱼ) pour UN couple → pas indépendantes, c'est terminé. Si la première case est OK, il faut vérifier toutes pour conclure indépendance.
🪤 0! = 1, pas 0
✓ Par convention mathématique
Cnn = n!/(n!×0!) = 1 car 0! = 1. Erreur fréquente dans le calcul des combinaisons.
🪤 E[X²] ≠ (E[X])²
✓ Var(X) = E[X²] − (E[X])² ≥ 0
E[X²] = Σ xᵢ²·P(xᵢ) (carrer les valeurs AVANT de multiplier par les probas). (E[X])² = carrer le résultat final de E[X].
🚀 Stratégie examen : (1) Lire tout l'énoncé. (2) Pour un tableau, toujours commencer par trouver la constante (Σ=1), puis les marges. (3) Dessiner l'arbre pour les proba conditionnelles. (4) Vérifier Σ=1 à chaque sous-question.

Mémento — Toutes les formules

── PROBABILITÉS DE BASE ───────────────────────────────────────
P(Ā)            = 1 − P(A)
P(A ∪ B)        = P(A) + P(B) − P(A ∩ B)
P(A|B)          = P(A ∩ B) / P(B)         [conditionnelle]
P(A ∩ B)        = P(A) × P(B)             [si A,B indép.]
P(A)            = Σ P(Eᵢ) × P(A|Eᵢ)      [prob. totales]
P(Eᵣ|A)        = P(Eᵣ)×P(A|Eᵣ) / P(A)   [Bayes]

── COMBINATOIRE ───────────────────────────────────────────────
Arrangement Aₙʳ  = n! / (n−r)!            [ordre, pas répét.]
Permutation Pₙ   = n!                      [tout ordonner]
Combinaison Cₙʳ  = n! / [r!(n−r)!]        [sans ordre]

── V.A. DISCRÈTE ──────────────────────────────────────────────
E[X]            = Σ xᵢ · P(X = xᵢ)
E[X²]           = Σ xᵢ² · P(X = xᵢ)
Var(X)          = E[X²] − (E[X])²
σ(X)            = √Var(X)
E[aX + b]       = a·E[X] + b
Var(aX + b)     = a²·Var(X)               [b disparaît !]

── COUPLES (X,Y) ─────────────────────────────────────────────
Indép.          : P(X=x, Y=y) = P(X=x)·P(Y=y) pour tout (x,y)
E[XZ]           = E[X]·E[Z]               [si indép.]
Var(X ± Z)      = Var(X) + Var(Z)         [si indép., tjrs +]
P(X=x|Y=y)     = P(X=x, Y=y) / P(Y=y)
Cov(X,Y)        = E[XY] − E[X]·E[Y]
r               = Cov(X,Y) / √[Var(X)·Var(Y)]

── LOIS USUELLES ─────────────────────────────────────────────
Bernoulli B(p)  : E=p, Var=p(1−p)
Binomiale B(n,p): P(X=k)=Cₙᵏpᵏqⁿ⁻ᵏ, E=np, Var=npq
Poisson Po(λ)  : P(X=k)=λᵏe⁻λ/k!,   E=Var=λ
